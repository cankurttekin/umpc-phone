# Operating System
### postmarketOS
Disadvantages:
- Touch does not work on redmi note 9s (Focaltech)
- not available for z flip, nothing phone 3a

Advantages:
- Good community

### Ubuntu Touch & Droidian
Disadvantages:
- Halium
- Read only system.
- Unstable
- Not available for z flip and nothing phone 3a

Advantages:
- ?

### LineageOS
Disadvantages:
- Needs more observing as umpc

Advantages:
- Stable
